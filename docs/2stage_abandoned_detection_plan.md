# 2段階アプローチ：遊休農地検出計画

## 背景・動機
- K-Means水田スクリーニング（第1段階）は完成済み（Acc 90-93%）
- 次の課題：水田以外の農地で「畑 vs 放棄地」を分類する
- 学会：リモセン学会（2026年秋）向け

## アプローチ概要

```
全筆データ（~10万筆）
  │
  ├─ 第1段階: K-Means（既存・教師なし）
  │    VH_minが最小のクラスタ → 水田と判定
  │    残り → 非水田農地（第2段階へ）
  │
  └─ 第2段階: 教師あり分類（新規構築）
       非水田農地に対して → 畑 or 放棄地
       モデル: RandomForest → 他モデルも試す
       教師データ: combined_labels.csv
```

### 教師データの注意点
- 全てGoogle Earth衛星画像からのAI（Claude）判読、現地検証なし
- 確信度「高」に絞ると放棄地39件のみに激減 → 「中」込みで使う
- 「草地化→放棄地」の判定が最も弱い（衛星画像1枚では限界）
- 論文では「衛星画像目視判読による教師データ（現地未検証）」と明記

### 耕作中の内訳問題
- 耕作中574件の中に田と畑が混在
- メモから推定：水田29件 / 畑181件 / 不明364件
- 特徴量で明確に分離可能（水田はVH_min低・dNDVI/dVH大）

## 特徴量パターン（田畑放棄地の比較）

| 特徴量 | 水田っぽい(29) | 畑っぽい(172) | 不明(372) | 放棄地(413) |
|--------|:---:|:---:|:---:|:---:|
| VH_min | **-22.8** | -20.7 | -19.9 | -19.0 |
| NDVI_flood | 0.41 | 0.44 | 0.51 | 0.59 |
| NDVI_grow | 0.70 | 0.51 | 0.59 | 0.66 |
| dNDVI | **0.29** | 0.07 | 0.08 | 0.07 |
| dVH | **-3.9** | -0.3 | -0.3 | -0.0 |

**重要な発見:**
- 水田はVH_min/dNDVI/dVHで明確に分離 → 第1段階K-Meansで対応済み
- **畑と放棄地の特徴量が近い** → 第2段階の本質的な課題
- 「不明」364件は特徴量が畑に近い → ただし正確性を重視して除外する方針

## 第2段階の設計

### 入力
- 第1段階で「非水田」と判定された筆
- 特徴量: VH_min, NDVI_grow, VH_winter, NDVI_flood, dNDVI, dVH（6特徴量）
  - 追加特徴量の検討余地あり（面積、NDVI時系列分散など）

### 教師データのフィルタリング
- 「除外」ラベル → 除去（森林・太陽光・集落は対象外）
- 「不明」メモの耕作中 → 除去（正確性を重視）
- 使用するのは：畑っぽい耕作中(~172件) + 放棄地(414件)
  - ※水田っぽい耕作中(29件)も除去（第1段階で水田判定されるため）

### モデル
- まずRandomForestでベースライン
- その後: GradientBoosting, SVM, LightGBM等も試す

### 評価方法
- 3地域のクロスバリデーション（地域をfoldとして使用）
- 指標: Accuracy, F1, Precision, Recall, IoU

## 既存コードとの関係
- `accuracy_check.py`: 第1段階のK-Meansパイプライン → そのまま活用
- `data/*.csv`: 特徴量CSV → 第2段階でも同じデータを使用
- `labels/combined_labels.csv`: 教師データ → 第2段階の訓練に使用
