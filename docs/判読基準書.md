# 目視判読基準書
## 耕作中畑 vs 耕作放棄地の2クラス分類

### 対象地域
- つくばみらい市（平野部）
- 稲敷市（平野部）
- 笠間市（丘陵部）

### 使用する画像
- **判読用**: Google Earth高解像度画像（0.5m程度）
- **補助**: Sentinel-2カラー画像（10m）
- **時系列確認**: Google Earthの過去画像スライダー（2〜3年分）

### 判読に使う季節
- **推奨**: 冬〜春（1〜4月）の画像を優先
- **理由**: 耕作中は裸地・耕起跡、放棄地は枯草・灌木が残り、差が明確
- **避ける**: 夏（7〜8月）は雑草と作物の区別が困難

---

## クラス定義

### クラス1: 耕作中（畑・果樹園を含む）

以下の**いずれか1つ**を満たす場合：

- [ ] 耕起跡・畝の筋が確認できる
- [ ] 均一な作物の被覆がある（列状・ブロック状）
- [ ] 果樹が等間隔に配置されている（栗園・梨園など）
- [ ] ハウス・マルチ等の施設が見える
- [ ] 過去2年以内の画像で上記いずれかが確認できる

### クラス2: 耕作放棄地

以下を**すべて**満たす場合：

- [ ] 管理の痕跡がない（耕起・刈取・除草なし）
- [ ] 不均一な雑草・灌木が繁茂している
- [ ] 周囲の耕作地と明らかに異なるテクスチャ
- [ ] 過去2年間の時系列画像でも管理痕跡がない

### 判定不能（除外）

以下に該当する場合はラベルを付けず除外する：

- 耕作中か放棄か判断できない（半放棄状態）
- 画像が雲・影で被覆されている
- 建物・道路等の非農地が混在している
- 画像の解像度が不足し判断できない

## 記録項目

各サンプルについて以下を記録する：

| 項目 | 内容 | 例 |
|------|------|-----|
| ID | 連番 | 001 |
| 地域 | 市名 | 笠間市 |
| 緯度 | 小数点以下6桁 | 36.345678 |
| 経度 | 小数点以下6桁 | 140.234567 |
| クラス | 耕作中 / 放棄地 / 除外 | 耕作中 |
| 確信度 | 高 / 中 / 低 | 高 |
| 判定根拠 | 何を見て判断したか | 畝の筋が明確 |
| 画像日付 | 判読に使った画像の撮影日 | 2022-03-15 |
| 地目参考 | eMAFF等の参考情報 | 畑（不耕作黄） |
| 判読者 | 誰が判読したか | Codex |
| 判読回 | 何回目の判読か | 1回目 |

### CSV実務運用

推奨CSV列は以下：

```csv
id,lat,lon,遊休農地フラグ,面積m2,住所,ラベル,確信度,メモ,画像日付,参照情報,判読者,判読回
```

既存CSVに列追加が難しい場合は、最低限 `メモ` に以下を追記する。

- `画像日付:YYYY-MM-DD`
- `参照:eMAFF不耕作黄 / 時系列確認あり / 参照なし`

メモ例:

- `畝明瞭; 画像日付:2022-03-15; 参照:eMAFF不耕作黄`
- `樹林化; 画像日付:2021-12-20; 参照:時系列確認あり`

## 確信度の運用

- `高`: 判読根拠が明確で、別の判読者でも同じ結論になりやすい
- `中`: ある程度判断できるが、画像条件や周辺環境の影響がある
- `低`: 耕作中と放棄地のどちらにも見え、断定しにくい
- `除外`: 判別不能。確信度は空欄でよい

### `低` を使うべき典型例

- 夏画像で作物と雑草の見分けが難しい
- 小区画で周辺の住宅や道路の影響が大きい
- 半放棄状態で管理痕跡が弱い
- 1時点の画像だけでは結論が出ない

`低` は積極的に使う。曖昧なサンプルを無理に `高` や `中` に寄せない。

## サンプリング計画

| 地域 | 耕作中 | 放棄地 | 合計（除外前） |
|------|--------|--------|---------------|
| つくばみらい市 | 150点 | 150点 | 300点 |
| 稲敷市 | 150点 | 150点 | 300点 |
| 笠間市 | 200点 | 200点 | 400点 |
| **合計** | **500点** | **500点** | **1,000点** |

- 笠間は不確実判定が多くなる見込みのため多めに設定
- 除外率20%で最終800点を目標
- 放棄地サンプルはeMAFF「不耕作黄」フラグのエリアを優先
- 第1段階ではランダム抽出で全体分布を確認する
- 第2段階では完全ランダム追加ではなく、放棄地候補を意図的にオーバーサンプリングする
- 第2段階候補は eMAFF フラグ、既存モデルの低確信度域、NDVI時系列の異常などから選ぶ
- 笠間市は平野部と別管理とし、除外多め・保守的判定を許容する

## 品質管理

1. 各市20〜30件は二重判読する
2. 2回目判読では1回目の結果を見ない
3. 一致率と不一致パターンを残す
4. 不一致例をもとに基準書を更新する
5. 学習データ作成時は、まず `高確信度` のみで初期モデルを作る
6. 次に `中`、`低` を段階的に追加し、性能とノイズ影響を比較する

## 注意事項

1. **時期の統一**: 衛星データ（2022年）と判読画像の年度を揃える。Google Earthで2022年前後の画像を使用
2. **最小面積**: 10m×10m未満の農地はSentinel解像度で判別不能のため除外
3. **エッジ回避**: 農地境界から10m以上内側のピクセルを使用
4. **判読順序**: 地域内でランダム順に判読し、バイアスを避ける
5. **休憩**: 連続100点以上の判読は判断精度が低下するため、1時間ごとに休憩
6. **笠間市の扱い**: 丘陵地・樹林隣接・地形影響が大きいため、平野部より保守的に判定する
