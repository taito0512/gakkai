## この作業の目的
衛星写真を見て、農地の筆を以下の3区分で目視ラベリングする。

- `耕作中`
- `放棄地`
- `除外`

このラベルは、将来的に「畑」と「耕作放棄地」を判別する教師あり機械学習モデルの訓練データになる。
現在のK-Means研究は水田スクリーニング用であり、この作業はその次段階のための正解データ作成である。

- クラスバランスは3市合算で取ればよく、市ごとに均等である必要はない

## 推奨する列追加
今後の学習と検証のため、可能ならCSVに以下の列を追加する。

- `画像日付`
  - 判読に使ったGoogle Earth画像の日付
  - 例: `2022-03-15`
- `参照情報`
  - 判読時に使った補助情報
  - 例: `eMAFF不耕作黄` / `時系列確認あり` / `参照なし`
- `判読者`
  - `Codex` / `ユーザー` など
- `判読回`
  - `1回目` / `2回目`

既存CSVの列をすぐ変えない場合は、少なくとも `メモ` に `画像日付:` と `参照:` を短く追記する。
例:

- `畝明瞭; 画像日付:2022-03-15; 参照:eMAFF不耕作黄`
- `雑草繁茂; 画像日付:2021-12-20; 参照:時系列確認あり`

## 判読ルール
判読の基準は `判読基準書.md` を優先する。要点は以下。

- `耕作中`
  - 耕起跡、畝の筋、均一な作物被覆、果樹の規則配置、ハウスやマルチなど管理痕跡がある
  - 過去2年以内の画像で管理状態が確認できれば耕作中としてよい
- `放棄地`
  - 管理痕跡がない
  - 不均一な雑草や灌木が繁茂
  - 周囲の耕作地と明らかに異なるテクスチャ
  - 過去2年間の画像でも管理痕跡がない
- `除外`
  - 森林内、住宅密集、道路沿い、太陽光パネル隣接、極小区画、非農地混在
  - 雲、影、低解像度などで判別不能
  - Sentinel解像度で扱いづらい極小農地

## 画像判読時の運用ルール
- ユーザーはGoogle EarthまたはGoogle Mapsのスクリーンショットを送る
- 画像には黄色ピンとIDが表示されている想定
- 1枚に複数ピンがある場合は、見えている全IDを判定する
- 可能なら冬から春の画像を優先する
- 夏画像しかない場合は、判定を保守的にし、必要なら `除外` または `低確信度` にする
- IDが見切れている場合は、座標や住所から推定する
- `遊休農地フラグ` は参考情報であり、最終判断は目視優先
- 稲敷市では eMAFF フラグを強く信用しない
- 笠間市は丘陵地のため、平野部より保守的に判定し、迷う場合は `低確信度` または `除外` を優先する
- 画像1枚だけで判定が不安定な場合は、可能なら過去画像で2年分程度を確認する

## 確信度の付け方
- `高`: 畝、耕起、樹林化、非農地化などが明確
- `中`: おおむね判断可能だが、画質、季節、面積、周辺混在の影響がある
- `低`: 耕作中か放棄地か判断に迷いが残る
- `除外` の場合は確信度を空欄でもよい

`低` は積極的に使うこと。無理に `高` や `中` に寄せない。

`低` を付ける典型例:

- 夏季画像で作物と雑草の区別が難しい
- 小区画で周辺の住宅や道路の影響が強い
- 半放棄の可能性がある
- 1時点では耕作中か放棄地か決めきれない

## メモの書き方
短く、判定根拠だけを書く。

例:
- `大区画褐色農地・耕起確認`
- `明確な畝列・野菜畑`
- `雑草繁茂・管理痕跡なし`
- `樹林化進行・不耕作黄参考`
- `森林内・農地として判別不能`
- `住宅密集エリア・判別不能`

## Codexへの期待動作
- 画像を見て各IDのラベル、確信度、メモを日本語で返す
- ユーザーが望む場合は、CSVの該当行へ反映まで行う
- CSV更新前に、対象ファイル名と更新対象IDを明示してから編集する
- 既存値を上書きする場合は、上書き対象を先に確認する
- 可能なら `画像日付` と `参照情報` も残す
- 判定に迷う場合は、無理に断定せず `低確信度` または `除外` を使う

## 品質管理
- 各市で20〜30件は二重判読を行い、一致率を確認する
- 二重判読では、1回目の結果を見ずに2回目を判定する
- 不一致例は `判読基準書.md` に追記し、ルールを更新する
- 学習用データは、まず `高確信度` のみで初期モデルを作る
- その後、`中` と `低` を加えた場合の性能変化を比較し、ノイズ耐性を確認する

## 現時点の注意
- 研究上はラベルの量だけでなく、一貫性と再現性が重要
- 画像日付を記録できるなら、今後の検証のため残した方がよい
- 笠間市は丘陵地で誤判定しやすいため、平野部より保守的に判定する

- 学習時は `高確信度のみ` と `高+中+低` の両方で比較する
